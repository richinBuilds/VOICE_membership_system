**Filtering Capabilities:**

**Backend Changes:**
- `AdminController.java` - Added comprehensive filtering:
  - **City Filter** - Partial text search (case-insensitive)
  - **Province Filter** - Exact match dropdown selection
  - **Chapter/Location Filter** - Searches child information records (partial text search)
  - New filter methods:
    - `filterByCity()` - Filters users by city (contains search)
    - `filterByProvince()` - Filters users by exact province match
    - `filterByChapterLocation()` - Filters users with children in specific chapter/location
  - All filters work together with existing filters (age, hearing loss type, equipment, date range)

**Frontend Changes:**
- `admin.html` - Enhanced filter panel:
  - Reorganized filters into responsive grid layout
  - Added **City** input field for text search
  - Added **Province** dropdown with all Canadian provinces/territories
  - Added **Chapter / Location** input field to search child chapter information
  - Filters retain values after submission for better UX
  - Updated table to display City and Province columns
  - All filters have clear labels and placeholders

**Display Improvements:**
- Added City and Province columns to admin user table
- Responsive table layout maintains readability
- "N/A" displayed for empty city/province fields

**Benefits:**
